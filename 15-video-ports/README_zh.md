*你可能想提前 Google 的概念：I/O 端口*

**目标：学习如何使用 VGA 卡数据端口**

我们将使用 C 通过 I/O 寄存器和端口与设备通信。

打开 `drivers/ports.c` 并检查内联 C 汇编语法。它有一些差异，比如源和目标操作数的顺序，以及将变量分配给操作数的有趣语法。

当你理解这些概念时，打开 `kernel/kernel.c` 查看一个使用示例。

在这个例子中，我们将检查映射屏幕光标位置的 I/O 端口。具体来说，我们将使用值 `14` 查询端口 `0x3d4` 以请求光标位置的高字节，并使用 `15` 查询同一端口以获取低字节。

当查询此端口时，它将结果保存在端口 `0x3d5` 中

不要错过使用 `gdb` 检查 C 变量值的机会，因为我们仍然无法在屏幕上打印它们。为此，为特定行设置断点，`breakpoint kernel.c:21` 并使用 `print` 命令检查变量。你现在不高兴我们花了一些时间编译交叉编译的 gdb 吗？;)

最后，我们将使用查询到的光标位置在该位置写入一个字符。
